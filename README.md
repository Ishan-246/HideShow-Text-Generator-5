# Hide/Show Text Generator - 5

A basic React toggle app that shows or hides a paragraph using a button.

##  Features
- Show/hide functionality
- Button toggles text
- Simple useState usage

##  Tech Stack
- React.js
- JavaScript
- CSS

##  How to Run
1. Clone this repo
2. Run `npm install`
3. Run `npm start`
